🎯 LO QUE VAS A HACER
═══════════════════════════════════════════════════════════════════

Vas a usar Google Colab (GRATIS) para compilar tu calculadora
en un APK que puedes instalar en Android.

NO necesitas instalar nada en tu PC ✅


📋 PREPARACIÓN (5 minutos)
═══════════════════════════════════════════════════════════════════

1. Abre Google Drive: drive.google.com
2. Crea carpeta: "CalculadoraAPK"
3. Sube estos archivos:
   • calculadora_android.py
   • buildozer.spec
   • icono.ico (opcional)


🚀 COMPILAR (40-60 minutos primera vez)
═══════════════════════════════════════════════════════════════════

1. Abre: colab.research.google.com
2. Nuevo cuaderno
3. Copia y pega este código:

┌────────────────────────────────────────────────────────────────┐
│ # Montar Drive                                                 │
│ from google.colab import drive                                 │
│ drive.mount('/content/drive')                                  │
│                                                                │
│ # Ir a carpeta                                                 │
│ %cd /content/drive/MyDrive/CalculadoraAPK                      │
│                                                                │
│ # Instalar herramientas                                        │
│ !apt-get update -qq                                            │
│ !apt-get install -y openjdk-11-jdk > /dev/null 2>&1           │
│ !pip install -q buildozer cython==0.29.33 kivy                 │
│                                                                │
│ # Compilar APK                                                 │
│ !buildozer android debug                                       │
│                                                                │
│ # Descargar APK                                                │
│ from google.colab import files                                 │
│ files.download('bin/calculadoraprofesional-2.0-debug.apk')    │
└────────────────────────────────────────────────────────────────┘

4. Ejecuta cada celda (Shift + Enter)
5. Espera... ☕ (30-60 minutos)
6. ¡Descarga tu APK!


📱 INSTALAR EN ANDROID
═══════════════════════════════════════════════════════════════════

1. Transfiere el APK a tu teléfono
2. Abre el archivo APK
3. Permite "Fuentes desconocidas"
4. Instala
5. ¡Listo! 🎉

🆘 SI ALGO FALLA
═══════════════════════════════════════════════════════════════════

Error común: "buildozer not found"
Solución: !pip install --upgrade buildozer

Error común: "Java not found"  
Solución: !apt-get install openjdk-11-jdk

Error común: Se detiene la compilación
Solución: Ejecuta de nuevo, continúa donde quedó
