📱 ANDROID - PRÓXIMOS PASOS
═══════════════════════════════════════════════════════════════════

Para crear el APK:

1. Instala Buildozer (Linux/WSL):
   pip install buildozer

2. Compila:
   cd android
   buildozer android debug

3. El APK estará en:
   android/bin/calculadoraprofesional-2.0-debug.apk

4. Instala en tu teléfono:
   - Copia el APK
   - Permite fuentes desconocidas
   - Instala

Alternativa: Usa la PWA desde la web
